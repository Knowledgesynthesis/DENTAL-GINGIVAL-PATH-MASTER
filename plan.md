# **7. ORAL PATHOLOGY REASONING FRAMEWORK**

1. Identify tissue type (epithelium, odontogenic, glandular, mesenchymal).  
2. Evaluate architecture (cystic vs solid, palisading, rosettes).  
3. Assess cytology (keratinization, atypia, pigment).  
4. Determine benign/reactive vs neoplastic.  
5. Apply IHC/molecular logic conceptually.  
6. Integrate patterns into broad diagnostic families.  
7. Review for pitfalls (SCC mimics, OKC lookalikes, pigmented lesion mimics).
